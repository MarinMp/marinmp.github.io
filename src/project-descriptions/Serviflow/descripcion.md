# Serviflow: Mini ERP de Servicios
Sistema integral de gestión para empresas de mantenimiento y reparación. Administra clientes, órdenes de servicio, técnicos y facturación con dashboard de métricas en tiempo real.

## Stack
Django, Next.js 15, PostgreSQL, Ant Design
