# HeyFood: Gestión de Dark Kitchens
Plataforma SaaS Multi-tenant para optimizar la operación de cocinas ocultas. Centraliza el menú digital, inventarios y pedidos en tiempo real con integración a WhatsApp y cálculo de domicilios por distancia.
