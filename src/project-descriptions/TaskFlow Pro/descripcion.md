# TaskFlow PRO: Gestión de Equipos
Sistema empresarial para la gestión de tareas orientado a equipos de trabajo. Permite organizar departamentos, asignar tareas y visualizar métricas de productividad por roles.

## Stack
Spring Boot 3.2, Angular 17, PostgreSQL, TailwindCSS
